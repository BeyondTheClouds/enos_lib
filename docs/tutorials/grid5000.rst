Tutorial 2 - Working with Grid'5000
===================================

Let's adapt the vagrant tutorial to Grid'5000.

Installation
------------

On Grid'5000, you can got with a virtualenv :

.. code-block:: bash

    $ virtualenv venv
    $ source venv/bin/activate
    $ pip install -U pip

    $ pip install enoslib
